## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Running the app

```bash
# desenvolvimento com live-reload
$ npm run start:dev

# produção
$ npm run start:prod
```

## Testandp

```bash
# testes de unidade
$ npm run test

# testes end to end
$ npm run test:e2e

# teste com cobertura
$ npm run test:cov
```
